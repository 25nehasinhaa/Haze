# Visual Direction

SignalRank should feel like a creative intelligence studio, not a generic ATS
dashboard.

## Reference Translation

- Dark canvas with high-contrast paper panels.
- Lime signal accents for verified evidence and key actions.
- Coral, cyan, yellow, and blue as secondary analysis colors.
- Floating layered cards to represent competing candidate signals.
- Dot-matrix patterns to echo signal processing and evidence density.
- Large editorial typography for judge memorability.
- Compact recruiter controls for real workflow usability.

## Palette

- Ink: `#070707`
- Panel: `#111111`
- Paper: `#f4f2ee`
- Signal lime: `#d9ff3f`
- Alert coral: `#ff5a3d`
- Analysis cyan: `#7bdcff`
- Ranking blue: `#4058ff`
- Caution yellow: `#ffd736`

## Interaction Principles

- First screen must explain the product without a paragraph-heavy landing page.
- Every visual flourish should map to the product concept: signal, noise,
  evidence, ranking correction, or hidden gems.
- Keep recruiter actions obvious: compare, inspect evidence, identify risk, and
  generate interview probes.
- Avoid decorative complexity that makes the ranking harder to understand.
